¿Qué te da listo?

Cada feature es un paquete independiente; puedes borrar lo que no uses sin romper nada. Las que dependen de una key opcional están marcadas.

[LLM]

Cascada con failover

Una sola llamada (complete_with_fallback) prueba Groq → OpenRouter → Gemini. Circuit breaker abre el proveedor caído por 60s. Tu usuario nunca se entera.

  • 4 cascadas: speed · reasoning · cheap · multimodal
  • 3 proveedores con planes gratuitos por default
  • Tests: 13 (failover + breaker + status shape)
[AGT]

Multi-agente con razonamiento visible

LangGraph (router → researcher → analyst → writer) + CrewAI (crews colaborativos). Cada paso emite un SSE; el panel lateral del chat lo dibuja en vivo.

  • 2 motores: graph (chat) · crew (colaborativo)
  • Tools inyectables: RAG, Web search
  • Multi-idioma: es / en / pt / fr / de / it
[RAG]

RAG con file adapter universal

Sube cualquier archivo y el agente lo usa. PDF · DOCX · XLSX · CSV · JSON · MD · HTML · TXT · Parquet · XML. La pipeline (chunk + SBERT + FAISS / pgvector) es compartida con los agentes.

  • FileAdapterFactory normaliza a texto + tabular
  • SBERT MiniLM-L6-v2 (384-dim) · sentence-aware chunks
  • Tests: 9 chunking + FAISS · 9 file factory
depende: ninguna API key · runs offline
[WEB]

Búsqueda web

Toggle use_web en el chat. Por default scraping HTML de DuckDuckGo (sin key). Con TAVILY_API_KEY sube de tier automáticamente.

  • DuckDuckGo HTML (resultados reales, sin Wikipedia-only)
  • Tavily como upgrade opt-in
  • Inyectable como tool en el orchestrator
depende: opcional TAVILY_API_KEY
[DB]

NL → SQL sobre tu DB

Conecta Postgres / MySQL / SQLite. /database/nl-query traduce lenguaje natural a SELECT seguro (DML/DDL bloqueado, multi-statement bloqueado, LIMIT auto).

  • SchemaIntrospector listo para LLM context
  • SafeQueryExecutor con regex de bloqueo
  • Tests: 12 (executor + introspector)
depende: tu DATABASE_URL_USER
[NOT]

Notificaciones multi-canal

El agente puede avisarte cuando algo pasa. Telegram + Email mandados en paralelo (asyncio.gather). Channels detectados al boot.

  • Severidades: info · warning · critical
  • Per-channel results · 6 tests con stubs
depende: TELEGRAM_BOT_TOKEN y/o SMTP
[PDF]

Exportar a PDF

Botón PDF en el aside del chat exporta la última respuesta. Doble pipeline: ReportLab por default (pure Python, funciona en Windows), WeasyPrint cuando hay GTK. Markdown del agente se renderiza con tablas, código, listas y links.

  • Datasheet PDF con pnpm datasheet
  • Reportes con métricas + tablas via WeasyPrint
[SET]

Listo para deploy

CI corre los 6 paquetes + Go build + 3-frontend matrix. CD auto-despliega Svelte / Next / Vue a Vercel y FastAPI / Go a Koyeb en cada merge a main.

  • Vercel · Koyeb · Supabase · PocketHost (todos free)
  • JWT auth opcional · rate limiter sliding-window

Empezar en 5 comandos

Asumiendo Python 3.12 y Node 20+. La primera vez tarda ~3 min por sentence-transformers.

# 1) clonar y configurar
git clone <repo> && cd agentos
cp .env.example .env  # pega GROQ_API_KEY (gratis en console.groq.com)

# 2) instalar packages
pnpm install
pip install -e packages/llm packages/agents packages/rag packages/files services/api

# 3) backend
cd services/api && uvicorn app.main:app --reload

# 4) elegir frontend
pnpm preview   # abre este selector visual

# 5) verificar
python scripts/e2e_smoke.py
